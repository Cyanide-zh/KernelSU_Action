# KernelSU Action

用于 Non-GKI Kernel 的 Action，具有一定的普遍性，需要了解内核及 Android 的相关知识得以运用。基于https://github.com/xiaoleGun/KernelSU_Action修改。

## 警告:warning: :warning: :warning:

如果你不是内核作者，使用他人的劳动成果构建KernelSU，请仅供自己使用，不要分享给别人，这是对原作者的劳动成果的尊重。

## 支持内核

- `4.19`
- `4.14`
